### Yotube Shortの視聴制限機能
Youtube Shortを見すぎて、時間を無駄にしてしまう方は使ってください。

### 使用法
1. Chrome右上の「︙」メニュー ＞「拡張機能」＞「拡張機能を管理」を選択し、右上にある「デベロッパーモード」をオンにしてください。
2. 「パッケージ化されていない拡張機能を読み込む」から、ダウンロードしたYoutubeShorts_Blockerのファイルを選択すると、機能が有効化されます。
3. Google_Chrome右上の「拡張機能」から、本拡張機能を固定していただき、クリックすると時間設定画面が表示されます。お好みの時間に設定してください。
4. Youtube_Shortの視聴が一日間、設定時間以内しかできなくなります。（設定時間を超過するとYoutubeShortのページ全体にぼかしがかかり、視聴が不可能になります）
5. 次の日の午前零時に、制限は解除されます。
